## Contenidos del repositorio

El repositorio está dividido en cinco prácticas principales:

### **Práctica 1 — Fundamentos de Series Temporales**
Código Python asociado al **Bloque 1** de la asignatura:

- Estacionariedad  
- ACF y AVF  
- Modelos básicos (Ruido Blanco, MA, AR)  
- Transformaciones y descomposición  

---

### **Práctica 2 — Modelos Estadísticos para Series Temporales**
Código Python asociado al **Bloque 2**:

- Identificación mediante ACF y PACF  
- Invertibilidad y causalidad  
- Ecuaciones de Yule–Walker  
- Metodología Box–Jenkins  
- Modelos ARIMA / SARIMA  

---

### **Práctica 3 — Análisis Espectral y Suavizado**
Código Python asociado al **Bloque 3A**:

- Periodograma y FFT  
- Método de Welch  
- Análisis espectral en modelos ARMA  
- Técnicas de suavizado temporal  

---

### **Práctica 4 — Predicción SARIMA**
Código Python asociado al **Bloque 3B**:

- Predicción mediante polinomios predictores  
- Error de predicción  
- Predicción en modelos SARIMA  
- Predicción en series reales  

---

### **Práctica 5 — Filtro de Kalman y Modelos VAR**
Código Python asociado al **Bloque 3C**:

- Implementación del filtro de Kalman  
- Predicción ARMA en espacio de estados  
- Estimación recursiva de parámetros  
- Predicción en datos reales  
- Modelos VAR con Kalman  

---
