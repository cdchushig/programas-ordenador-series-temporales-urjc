Código fuente de asignatura Análisis de Series Temporales
====

Este repositorio contiene el código fuente para la asignatura de Análisis de Series Temporales del 
Grado en Ciencia e Ingeniería de Datos de la Universidad Rey Juan Carlos, el cual es presentado
para la convocatoria de asignaturas en abierto del curso 2025-2026. El repositorio esta divido en 5 prácticas:

### Contenidos del repositorio

1. **Práctica 1.** Código Python asociado al bloque 1 de la asignatura: Fundamentos de series temporales.
2. **Práctica 2.** Código Python asociado al bloque 2 de la asignatura: Modelos estadísticos para series temporales.
3. **Práctica 3.** Código Python asociado al bloque 3a de la aignatura: Filtrado, suavizado, predicción y filtrado de Kalman.
4. **Práctica 4.** Código Python asociado al bloque 3b de la asignatura: Filtrado, suavizado, predicción y filtrado de Kalman.
4. **Práctica 4.** Código Python asociado al bloque 3c de la asignatura: Filtrado, suavizado, predicción y filtrado de Kalman.
